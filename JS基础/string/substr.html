substr

定义和用法

从指定位置，抽取指定数量的字符
substr() 方法可在字符串中抽取从 开始 下标开始的指定数目的字符。 提示： substr() 的参数指定的是子串的开始位置和长度，因此它可以替代 substring() 和 slice() 来使用。 注意： substr() 方法不会改变源字符串。

语法

string.substr(start,length)

参数

start 必需。要抽取的子串的起始下标。必须是数值。包含当前下标元素。 如果是负数，那么该参数声明从字符串的尾部开始算起的位置。也就是说，-1 指字符串中最后一个字符，-2 指倒数第二个字符，以此类推。 length 可选。

子串中的字符数。
必须是数值。如果省略了该参数，那么返回从 stringObject 的开始位置到结尾的字串

返回值

String A new string containing the extracted part of the text
