charAt

返回字符串指定位置的字符，如果没有则返回一个 '' 

'asd'.charAt(1) // 's'
'asd'.charAt(5) // ''
'asd'.charAt() // 'a'

charCodeAt

返回字符串指定位置字符的 ASCII 编码

'd'.charCodeAt(2) // NaN
'd'.charCodeAt(0) // 100
'd'.charCodeAt() // 100

fromCharCode

返回 ASCII 编码对应的字符

String.fromCharCode(100) // 'd'
String.fromCharCode() // ''

综合应用

        String(2).charCodeAt() // 50
        String.fromCharCode(50) // "2"
        String(2).charAt() // "2"
