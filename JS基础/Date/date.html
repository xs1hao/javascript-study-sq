Date 对象方法 
            new Date() 实例上的方法只有 getDate() 的返回值是从 1 开始的，其他的都是从 0 开始的；
            var myDate = new Date('2022-01-01')；
            myDate.getDate(); // 1；从 1 开始；
            myDate.getMonth(); // 0；
            getDay() // 返回星期几，默认星期天为 0 ；星期六为 6；

同时 获取一个完整的年份，应该使用的是 myDate.getFullyear(); 而不是 myDate.getYear();
