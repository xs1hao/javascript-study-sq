sort()

定义和用法

支持number 数字型 和 字符串的排序 sort()方法用于对数组的元素进行排序。 排序顺序可以是字母或数字，并按升序或降序。 默认排序顺序为按字母升序。

注意：当数字是按字母顺序排列时"40"将排在"5"前面。 使用数字排序，你必须通过一个函数作为参数来调用。 函数指定数字是按照升序还是降序排列。

sort() 函数解读：
1）一个数组进行sort() 排序，a,b 如何取值？
sort(a,b) 接收两个参数，a是从第1个，b 是从第 0 个开始取； sort() 函数怎么排序，每次读取的两个参数，是什么规则？
关于js数组sort方法的实现原理总结 2）sort 工作原理: 第一个值应该在第二个值 之后 的情况下，返回1，
而第一个值应该在第二个值的 前面 的情况返回 -1；不做位置更换，则返回0 这里的意思是
1、如果 a < b 返回了 1 (第一个值应该在第二个值之后的情况下) ===>> 等价于，小的值，放在后面；也就是我们常说的降序排序； 2、如果 a > b 返回了 1 (第一个值应该在第二个值之后的情况下) ===>> 等价于，大的值，放在后面，也就是成 升序排序； 3、同理，当返回 -1 时，是相反的道理；a < b 返回 -1 表示，a 应该放在 b 的前面，升序；a > b 返回 -1， 大的值放在前面，降序；

语法

function compare(a,b) {
  if( a < b ){
    return -1;
  }else if( a > b ){
    return 1;
  }else {
     return 0;
  }
}
array.sort(compare);
